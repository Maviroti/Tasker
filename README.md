# Tasker

Платформа для управления задачами.
Задачи разделены на базовые типы (TASK, BUG, FEATURE, PBI, EPIC).
У задач есть статус (ACTIVE, CLOSED, NEW), с помощью которого можно просматривать задачи в стиле канбанов.
Для задач можно задавать теги.

В проекте реализована регистрация и авторизация пользователей. Для создания задачи обязательным полем является `Пользователь`, поэту в системе должен быть хотя бы один пользователь. Возможность регистрации пользователей позволяет работать с проектом в команде.
Возможность удалять задачи доступна только для пользователей с правами администратора.

## Зависимости

Все зависимости собраны через `poetry`.
Установка зависимостей:

```bash
pip install poetry
poetry install
```

## База данных

Для проекта используется БД `postgreSQL`
Для настройки подключения к БД, необходимо скопировать параметры из `.env-example` в новый файл `.env` и заполнить поля.

## Запуск

Проект написан на Jdango.
Для запуска проекта нужно запустить:

```bash
# Накат миграций БД
python manage.py migrate
# Запуск сервера
python manage.py runserver
```

## Celery workers

Реализована задача по логированию новых задач в консоль

### Запуск workers

```bash
celery -A config worker --loglevel=info
```

для Windows

```powershell
celery -A config worker --loglevel=info --pool=solo
```
